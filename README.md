# Search Engine Project

A complete search engine implementation with web crawling, indexing, and query processing capabilities.

## Features

- **Web Crawler**: Scrapy-based crawler with politeness controls
- **Indexer**: TF-IDF with cosine similarity and optional FAISS
- **Query Processor**: Flask-based API with spell checking and query expansion
- **Web Interface**: Modern, responsive search interface
